Proyectos

Metrosport - Ligas Urbanas

Aplicación para crear y unirse a ligas urbanas de deportes. Desarrollada con Laravel y Vue.js, permite registrarse como equipo, inscribirse en distintas ligas y comunicarse mediante un chat interno. Incluye inteligencia artificial para comparar automáticamente las disponibilidades horarias entre equipos y facilitar la organización de partidos.


metrosport.ainhoablanco.me

ChillGig - Gestión de eventos

Aplicación web creada por un equipo de cuatro personas que conecta músicos y locales. Permite registrarse, buscar perfiles mediante filtros, chatear, ver contenido multimedia dentro del perfil y consultar un mapa con las ubicaciones. Desarrollada con Laravel y Vue.js.

AinTask- Gestión de tareas

Herramienta de gestión de proyectos desarrollada con JavaScript, HTML, CSS y PHP. Permite crear proyectos, añadir tareas dentro de ellos y asignarlas a los miembros correspondientes, facilitando la organización del trabajo.


einagestio.ainhoablanco.me

Cientifiks - Juego

Página web desarrollada en grupo por cinco personas como proyecto educativo. Consta de una landing page y cinco juegos interactivos, uno por cada miembro del equipo. El quinto juego, creado por mí, está programado con JavaScript, HTML y CSS.
